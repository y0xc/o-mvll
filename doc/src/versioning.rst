Versioning
~~~~~~~~~~

.. py:attribute:: omvll.LLVM_VERSION

   The version of LLVM used to compile O-MVLL.
   For instance ``21.0.0``

.. py:attribute:: omvll.OMVLL_VERSION

   The current O-MVLL version (e.g. ``1.8.0``)

.. py:attribute:: omvll.OMVLL_VERSION_FULL

   The current O-MVLL full version including the LLVM commit.
   For instance: ``OMVLL Version: 1.8.0 / 21.0.0 (4c603efb0cca074e9238af8b4106c30add4418f6)``
